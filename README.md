# 🛒 MultiCart – Full Stack E-Commerce Platform

MultiCart is a modern **full-stack e-commerce web application** that allows users to browse products, manage carts, and place orders seamlessly.
It is built with modern web technologies and designed with scalability and clean UI in mind.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/multicart.git
```

### 2️⃣ Navigate to project

```
cd multicart
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Create `.env` file

```
MONGODB_URI=your_mongodb_connection_string
NEXTAUTH_SECRET=your_secret
NEXTAUTH_URL=http://localhost:3000
```

### 5️⃣ Run the development server

```
npm run dev
```

Open

```
http://localhost:3000
```
